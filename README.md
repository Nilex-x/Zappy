# Zappy

## Zappy is simulation of life

# instalation
``` bash
make
```

# launch server
``` bash
./zappy_server -p [PORT] -x [WIDTH] -y [HEIGHT] -n ([TEAM_NAME], ...) -c [MAX_PER_TEAM] -f [FREQUENCY]
```

# launch IA
``` bash
./zappy_ai -p [PORT] -n [NAME_TEAM] -h [IP_SERVER]
```

# launch GUI
``` bash
./bin/zappy.x86_64
```
